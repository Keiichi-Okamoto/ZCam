# 📷 AIとSwiftUIでカメラアプリを作る

> Claude（AI）とのPRのやり取りを通じてiOSカメラアプリを
> 段階的に開発するプロジェクトです。

## このリポジトリについて

SwiftUI と AVFoundation を使ったiOSカメラアプリを、AI（Claude）と共同開発するプロジェクトです。
コードの実装はすべてAIが担当し、開発者は仕様の定義とPRレビュー・指示のみを行います。

開発プロセスそのものをZennの連載記事として公開しており、
GitHubのPR履歴を追うことで「AIとどうやって開発を進めるか」が分かる構成になっています。

## 連載記事

Zennの「本」と連動しています。
👉 [AIとSwiftUIでカメラアプリを作る]（ZennのURL）

## 開発環境

- Xcode 26.x
- Swift 6
- iOS 17以降
- 実機必須（カメラ機能のためシミュレーター不可）

## 実機で動かす方法

1. リポジトリをクローン
2. `Signing & Capabilities` の `Team` を
   ご自身のApple IDに変更
3. 実機を接続してRunを実行

## 開発の進め方

FEATURES.mdに機能仕様を定義し、AIがPRを作成 → レビュー → 修正、
というワークフローで開発しています。

**コードの実装・修正はすべてAIが行います。**
開発者はFEATURES.mdへの仕様記述とPRレビュー・指示のみを担当します。

また、以下のClaudeを目的に応じて使い分けて連携しています。

| ツール | 用途 |
|--------|------|
| claude.ai（Chat） | 企画・相談・ドキュメント整理・記事構成 |
| Claude Code（CLI） | コード生成・ビルド・技術的作業 |
| Cowork | PR作成・ファイル操作 |
